# ウィジェット単語アプリ 共有用まとめ

> ⚠️ この資料は `PRODUCT_SPEC.md`（統合版・唯一の正）に統合されました。確定方針はそちらを参照してください。
> 統合時の変更: 期限スケジューリングと手動「覚えた/まだ」は不採用（受け身の表示回数メーターに一本化）、音声は Phase 1 に前倒し。フォルダ主役・手動登録は採用。

最終更新: 2026-06-19

この資料は、別セッションで構成検討を続けるための共有用メモ。

実装に入るための仕様書ではなく、「ここまでで決まった方向性」と「次に詰めること」を渡すためのデータとして使う。

## 1. 現時点の芯

### コンセプト

覚えたい言葉を、毎日の画面に置いておくアプリ。

ユーザーがアプリを開いて勉強するのではなく、ホーム画面やロック画面を見るたびに、単語・読み方・意味が自然に目に入る状態をつくる。

### 一言でいうと

タップしなくても少しずつ覚えられる、ながら見の単語ウィジェット。

### 大事な差別化

- 単語・読み方・意味が最初から見えている
- タップして回答を出すクイズ型を主役にしない
- 英語だけに限定しない
- 学習アプリっぽすぎない、静かでニュートラルな見た目にする
- 通知、ランキング、連続記録で急かさない

## 2. 重要な方針変更

### セットではなくフォルダを主役にする

代表セットやプリセットを選ばせるアプリにはしない。

主役は、ユーザーが自由に作れるフォルダ。

例:

- 韓国旅行で使う単語
- 英語会議
- デザイナー英語
- カフェで使う言葉

サンプルは使い方を伝えるために入れるが、アプリの中心にはしない。

### サンプルフォルダ

Phase 1 では「韓国旅行で使う単語」をサンプルとして使うのがよい。

理由:

- 旅行前という目的がわかりやすい
- 期限設定との相性がよい
- App Store のスクリーンショットで価値が伝わりやすい
- ユーザーが自分のフォルダを作る参考になる

## 3. Phase 1 で届けたい状態

ユーザーが覚えたい言葉をフォルダに登録し、ホーム画面またはロック画面にウィジェットを置く。

その後は、アプリを開かなくても毎日数語が目に入り、覚えた / まだ の軽い記録だけで復習が回る。

Phase 1 の成功条件:

- 初回起動から3分以内にウィジェットを置ける
- ウィジェットだけで単語・読み方・意味がわかる
- 自由にフォルダを作って単語を登録できる
- 例フォルダだけでも使い方がわかる
- いつまでに覚えたいかを指定できる
- 指定した期限に合わせて、出る回数や頻度が自然に変わる

## 4. 初回体験

初回は聞きすぎない。

流れ:

1. コンセプトを短く見せる
2. 最初のフォルダを作る、または例フォルダを選ぶ
3. いつまでに覚えたいかを選ぶ
4. まず数語だけ登録する、または例フォルダで試す
5. 今日の単語を見せる
6. ウィジェット追加を案内する
7. ホーム画面へ戻る

初回で聞かないもの:

- 学習目標
- 1日の学習時間
- レベル診断
- 通知許可
- アカウント登録

## 5. フォルダ作成

### 作成フロー

1. フォルダ名を入れる
2. いつまでに覚えたいかを選ぶ
3. おすすめ単語数を見る
4. 最初の3語を登録する、または例フォルダを試す
5. ウィジェットの見え方を確認する

### 入力項目

| 項目 | 扱い | 見せ方 |
|---|---|---|
| フォルダ名 | 必須 | 例: 韓国旅行で使う単語 |
| 期限 | 必須 | 具体日 / 期間 / 期限なし から選ぶ |
| 説明 | 任意 | 何のためのフォルダかをメモできる |
| アイコン / 色 | 任意 | フォルダを見分けやすくする |
| 表示ペース | 詳細設定 | 基本は自動。必要な人だけ変更できる |

最初の作成画面では、フォルダ名と期限だけを目立たせる。

説明、アイコン、色、表示ペースは後から変更できる設定にする。

## 6. 期限とおすすめ単語数

「記憶曲線」や「復習間隔」という言葉は画面に出さない。

画面上は次のように聞く。

- いつまでに覚えたいですか？
- この期限なら、まず10語くらいがおすすめです
- 期限が近いので、よく使う言葉から登録しましょう
- あとから単語はいつでも追加できます

### 期限の入力方法

| 入力方法 | 向いている使い方 | 表示例 |
|---|---|---|
| 具体日 | 旅行、試験、発表など日付が決まっている | 7月20日まで |
| 期間 | ゆるい学習、仕事用語、趣味 | 1週間後まで / 1か月後まで |
| 期限なし | いつか覚えたい言葉を貯める | 期限なしでゆっくり |

旅行系フォルダでは、具体日を「出発日」として見せると自然。

### おすすめ単語数

| 期限 | おすすめ単語数 | 見せ方 |
|---|---:|---|
| 3日後まで | 5-10語 | まずは本当に使う言葉だけ |
| 1週間後まで | 10-20語 | 旅行前に覚える短期フォルダ |
| 2週間後まで | 20-30語 | よく使う言葉を広めに |
| 1か月後まで | 30-50語 | 少しずつ覚える標準フォルダ |
| 期限なし | 上限なし | ゆっくり増やせる |

登録数は基本的に強制制限しない。

多すぎる場合は、登録を止めるより、軽い注意と優先表示にする。

例:

- 30語あります。全部出すより、よく使う言葉から優先します
- 期限が近いので、重要な単語を先に表示します

## 7. 画面構成

Phase 1 の画面:

| 画面 | 役割 |
|---|---|
| ホーム | 今日の単語、選択中フォルダ、進捗を見せる |
| フォルダ一覧 | 作成したフォルダを管理する |
| フォルダ作成 | 名前、期限、登録数の目安を決める |
| 単語登録 | 単語、読み方、意味、例文を登録する |
| 単語詳細 | 例文、使う場面、状態を確認する |
| ウィジェット案内 | ウィジェット追加方法を案内する |
| 設定 | 表示数、更新タイミング、テーマなどを最小限で扱う |

タブは増やしすぎない。

推奨:

1. ホーム
2. フォルダ
3. 設定

## 8. ウィジェット方針

共通方針:

- 単語・読み方・意味は常に見えている状態にする
- タップして答えを表示するクイズ型にはしない
- 軽い操作だけ入れる

サイズ別の方向:

| サイズ | 目的 | 表示 |
|---|---|---|
| 小 | 一瞬で1語だけ目に入る | 単語、読み方、短い意味 |
| 中 | 1語を少し深く見られる | 単語、読み方、意味、例文1行、覚えた / まだ |
| 大 | 今日の3単語をまとめて見る | 3単語、それぞれの読み方、短い意味、状態 |
| ロック画面 | 見るだけの接触回数を増やす | 単語、読み方、短い意味 |

## 9. 復習の考え方

複雑な設定画面は入れない。

ただし、同じ単語が記憶に残りやすい間隔で何度か出るようにする。

画面上では「復習スケジュールを自動で調整しています」くらいに見せる。

優先して出す単語:

1. 期限が近い
2. まだ
3. 表示回数が少ない
4. 未確認
5. 覚えた

期限に対して単語数が多すぎるフォルダでは、全部を均等に出さない。

アプリ側で「まず覚える単語」を優先して出す。

## 10. Phase 1 から外すもの

- AI自動生成
- 音声再生
- ランキング
- 友達共有
- 学習時間記録
- 大量のテストモード
- 複雑な復習間隔の手動設定
- PDF / 画像 / ノートからの生成
- アカウント必須化

## 11. App Store で伝えること

伝える順番:

1. 覚えたい言葉を、毎日の画面に。
2. 単語・読み方・意味がいつも見える。
3. 旅行や仕事に合わせてフォルダを作れる。
4. ウィジェットで、アプリを開かずに復習できる。
5. 学習アプリっぽすぎない静かな見た目。

スクリーンショット構成案:

1. ホーム画面とウィジェットの並び
2. 小・中・大ウィジェットの比較
3. フォルダ作成と期限設定
4. 韓国旅行で使う単語フォルダの使用例
5. 覚えた / まだ の軽い復習

## 12. 次に決めること

優先度が高いもの:

1. アプリ名とApp Store上の見せ方
2. ウィジェット小・中・大の具体レイアウト
3. ホーム画面に出す情報量
4. サンプルフォルダを初期データとして入れるか、作成例として見せるだけにするか
5. 「覚えた / まだ」をウィジェット上で直接押せる体験にするか
6. ロック画面ウィジェットをPhase 1必須にするか

次の検討順:

1. アプリ名候補を出す
2. ウィジェット3サイズのワイヤーを決める
3. ホーム画面の情報量を決める
4. 「韓国旅行で使う単語」フォルダを初期データにするか決める
5. App Store スクリーンショットの見せ方を決める

## 13. 別セッションへの注意

- 実装にはまだ入らない
- 「代表セット」や「プリセット選択」を中心に戻さない
- フォルダ自由作成を中心にする
- 例フォルダは使い方を伝えるための補助として扱う
- 画面文言では「記憶曲線」「アルゴリズム」などの専門語を出さない
- Phase 1 ではAI生成と音声は外す
