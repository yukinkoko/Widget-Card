# ウィジェット単語アプリ Phase 1 構成案

## 1. このアプリの芯

### コンセプト

覚えたい言葉を、毎日の画面に置いておくアプリ。

ユーザーがアプリを開いて勉強するのではなく、ホーム画面やロック画面を見るたびに、単語・読み方・意味が自然に目に入る状態をつくる。

### 一言でいうと

タップしなくても少しずつ覚えられる、ながら見の単語ウィジェット。

### 大事にすること

- 単語・読み方・意味が最初から見えている
- 勉強を急かさない
- 通知やランキングで煽らない
- 英語だけに限定しない
- 目的別の言葉セットを主役にする
- ウィジェットを置いた瞬間から価値が出る

### やらないこと

- 回答を隠すクイズ型を主役にしない
- テストモードを複雑にしない
- 学習時間や連続記録でプレッシャーをかけない
- 手入力単語帳アプリに寄せすぎない
- Phase 1ではAI生成や音声を入れない

## 2. Phase 1 のゴール

### ユーザーに届けたい状態

ユーザーが目的別セットを選び、ホーム画面またはロック画面にウィジェットを置く。  
その後は、アプリを開かなくても毎日数語が目に入り、覚えた / まだ の軽い記録だけで復習が回る。

### Phase 1 の成功条件

- 初回起動から3分以内にウィジェットを置ける
- ウィジェットだけで単語・読み方・意味がわかる
- 覚えていない単語が後日また出る
- いつまでに覚えたいかを指定できる
- 指定した期限に合わせて、出る回数や頻度が自然に変わる
- プリセットだけでも使い始められる
- 手動登録はできるが、主役に見えない

## 3. 想定ユーザー

### 優先ユーザー

1. 旅行前に少し言葉を覚えたい人
2. 英語や韓国語をゆるく続けたい人
3. 仕事や制作で使う言葉を日常的に見ておきたい人

### Phase 1 で一番狙うユーザー

旅行前の実用層。

理由:

- 目的がはっきりしている
- 使う場面を想像しやすい
- プリセットセットとの相性が良い
- App Store のスクリーンショットで価値が伝わりやすい

## 4. アプリ全体の構成

### 画面一覧

| 画面 | 役割 | Phase 1 必須 |
|---|---|---|
| ホーム | 今日の単語、選択中セット、進捗を見せる | 必須 |
| セット選択 | 目的別プリセットを選ぶ | 必須 |
| 単語詳細 | 例文、使う場面、状態を確認する | 必須 |
| ウィジェット案内 | ウィジェット追加方法を案内する | 必須 |
| 手動追加 | 単語を追加・編集する | 小さく入れる |
| 設定 | 表示数、更新タイミング、テーマなど | 最小限 |

### タブ構成案

Phase 1 はタブを増やしすぎない。

推奨:

1. ホーム
2. セット
3. 設定

単語詳細や手動追加は、タブではなく各画面から遷移する。

## 5. 初回体験

### 初回起動の流れ

1. コンセプトを短く見せる
2. 目的別セットを選ばせる
3. いつまでに覚えたいかを選ばせる
4. 今日の単語を見せる
5. ウィジェット追加を案内する
6. ホーム画面へ戻る

### 初回で聞きすぎないこと

- 学習目標
- 1日の学習時間
- レベル診断
- 通知許可
- アカウント登録

初回は「セットを選んでウィジェットを置く」だけに寄せる。

### 期限設定の見せ方

「記憶曲線」や「復習間隔」という言葉は出さず、次のように簡単に聞く。

- 1週間後まで
- 2週間後まで
- 1か月後まで
- 期限なしでゆっくり

旅行セットの場合は「出発日」を入れられるとわかりやすい。

Phase 1 では、期限は単語ごとではなくセット単位で持つのがおすすめ。

例:

- 韓国旅行セット: 7月20日までに覚えたい
- 英語会議セット: 1か月かけて覚えたい
- デザイナー英語セット: 期限なしでゆっくり

## 6. ホーム画面構成

### 表示する内容

1. 今日の単語
2. 選択中のセット
3. 覚えた / まだ
4. ウィジェットの見え方
5. 簡易進捗

### ホームでやること

- 今日の単語を確認する
- 覚えた / まだ をつける
- 詳細を見る
- セットを切り替える
- ウィジェット追加方法を見る

### ホームでやらないこと

- 大量の単語一覧を見せない
- 学習時間グラフを大きく出さない
- クイズ開始ボタンを主役にしない
- AI生成導線を主役にしない

## 7. ウィジェット構成

### 共通方針

単語・読み方・意味は常に見えている状態にする。  
タップして答えを表示するクイズ型にはしない。

### 小サイズ

目的:

一瞬で1語だけ目に入る。

表示:

- 単語
- 読み方
- 短い意味

操作:

- タップで単語詳細を開く

### 中サイズ

目的:

1語を少し深く見られる。

表示:

- 単語
- 読み方
- 意味
- 例文1行
- 覚えた / まだ

操作:

- 覚えた
- まだ
- 詳しく見る

### 大サイズ

目的:

今日の3単語をまとめて見る。

表示:

- 今日の3単語
- それぞれの読み方
- それぞれの短い意味
- それぞれの状態

操作:

- 各単語の詳細
- 可能なら覚えた / まだ

### ロック画面

目的:

さらに軽く、見るだけの接触回数を増やす。

表示:

- 単語
- 読み方
- 短い意味

操作:

- タップで詳細へ

## 8. 単語データの構成

### 1セットが持つ情報

| 項目 | 必須 | 用途 |
|---|---|---|
| セット名 | 必須 | 例: 韓国旅行 |
| 説明 | 必須 | 何のための言葉か |
| 目標日 | 任意 | いつまでに覚えたいか |
| 期限タイプ | 必須 | 具体日 / 期間 / 期限なし |
| 表示ペース | 必須 | 標準 / 集中 / ゆっくり |
| 単語一覧 | 必須 | セットに含まれる単語 |

### 1単語が持つ情報

| 項目 | 必須 | 用途 |
|---|---|---|
| 単語 | 必須 | メイン表示 |
| 読み方 | 必須 | 発音の補助 |
| 意味 | 必須 | ウィジェット表示 |
| 例文 | 必須 | 中サイズ、詳細 |
| 使う場面 | 必須 | 旅行・仕事などの文脈 |
| カテゴリ | 必須 | セット内の整理 |
| 難易度 | 任意 | Phase 2 以降でもよい |
| メモ | 任意 | 手動登録向け |
| 状態 | 必須 | 覚えた / まだ |
| 表示回数 | 必須 | 何回ウィジェットに出たか |
| 最終表示日 | 必須 | 次に出すタイミングの判断 |
| 最終回答日 | 必須 | 覚えた / まだ を押した日 |

### 状態

- 未確認
- まだ
- 覚えた

Phase 1 では、この3つで十分。

## 9. 復習ロジック

### 基本方針

複雑な設定画面は入れない。  
ただし、記憶に残りやすい間隔で同じ単語が何度か出るようにする。

### Phase 1 のおすすめルール

- 覚えた: 少し間隔を空けてまた出す
- まだ: 近いうちにまた出す
- 未確認: 通常の候補として出す
- 期限が近い単語: 出る頻度を少し上げる
- 表示回数が少ない単語: 優先して出す

### 期限に合わせた出し方

ユーザーが「いつまでに覚えたいか」を指定すると、アプリ側で出す回数を調整する。

例:

| 期限 | 出し方 |
|---|---|
| 1週間後まで | 毎日同じ単語が複数回出てもよい |
| 2週間後まで | まだの単語を短い間隔で出す |
| 1か月後まで | 覚えた単語も数日おきに再表示する |
| 期限なし | ゆっくり全体を回す |

旅行セットでは、出発日から逆算して「よく使う単語」を優先する。

### 表示の考え方

今日の単語は、選択中セットから3語を出す。

優先順:

1. 期限が近い
2. まだ
3. 表示回数が少ない
4. 未確認
5. 覚えた

### 見せ方の注意

ユーザーには「復習スケジュールを自動で調整しています」くらいに見せる。  
細かい間隔やアルゴリズム名は出さない。

## 10. 初期プリセット案

### Phase 1 で入れる5セット

1. 韓国旅行で使う言葉
2. 海外旅行の空港・ホテル・レストラン
3. 英語の会議でよく使う言葉
4. デザイナーが覚えておくと便利な英語
5. カフェ・買い物・道案内

### 各セットの語数

初期は各20語を推奨。

理由:

- 少なすぎるとすぐ見飽きる
- 50語以上だと初期制作と確認が重くなる
- 20語なら品質確認しやすい

### セットの見せ方

手入力よりも、目的別教材に見えるようにする。

例:

- 韓国旅行
- 英語会議
- デザイナー英語

「単語帳1」「単語リスト」のような名前は避ける。

## 11. 手動登録の扱い

### 方針

手動登録は残すが、主役にしない。

### できること

- 単語を追加する
- 読み方を入れる
- 意味を入れる
- 例文を入れる
- セットに追加する

### 見せ方

ホームの主導線には置かず、セット詳細や設定付近から入る。

## 12. Phase 1 から外すもの

- AI自動生成
- 音声再生
- ランキング
- 友達共有
- 学習時間記録
- 大量のテストモード
- 複雑な復習間隔の手動設定
- PDF / 画像 / ノートからの生成
- アカウント必須化

## 13. App Store で伝えること

### 伝える順番

1. 覚えたい言葉を、毎日の画面に。
2. 単語・読み方・意味がいつも見える。
3. 旅行や仕事に合わせてセットを選べる。
4. ウィジェットで、アプリを開かずに復習できる。
5. 学習アプリっぽすぎない静かな見た目。

### スクリーンショット構成案

1. ホーム画面とウィジェットの並び
2. 小・中・大ウィジェットの比較
3. 目的別セット選択
4. 単語詳細
5. 覚えた / まだ の軽い復習

## 15. 現時点のおすすめ結論

Phase 1 は「韓国旅行」セットを最初の代表体験にする。

理由:

- 目的が具体的
- 旅行前という使うタイミングが明確
- 単語・読み方・意味の必要性が伝わりやすい
- ウィジェットで毎日見る価値がわかりやすい

まずは韓国旅行セットで、ホーム画面・小ウィジェット・中ウィジェット・大ウィジェットの具体デザインを固める。  
その後、他のセットへ横展開する。
