# claude.ai/design 引き継ぎメモ

claude.ai/design（ブラウザ）で画面の細かい調整をするための持ち込みセット。

## 使い方
1. `docs/mockups/png/` の画像を claude.ai/design にドラッグ＆ドロップ（または貼り付け）
2. 下の「貼り付け用プロンプト」をコピペして文脈を渡す
3. 画面ごとに「余白・文字サイズ・配置・トーンを整えて」と指示
4. 詳細仕様が要るときは `docs/PRODUCT_SPEC.md` を参照

## 貼り付け用プロンプト（コピペ）

> ウィジェット単語アプリ（アプリ名は未定）の画面を微調整したいです。添付画像がたたき台です。
>
> コンセプト: 覚えたい言葉をホーム/ロック画面ウィジェットに常時表示し、タップしなくても“ながら見”で覚える。学習アプリっぽさは抑える。
>
> ビジュアル方針（厳守）: 完全モノクロ（白・薄グレー・墨）。差し色なし、操作/アクティブ状態だけ濃いインク。サンセリフ統一・ウェイトで階層。静か・大人っぽい・毎日見ても疲れない。ただし“ちゃんとしたアプリ”の作り込み（カード・タブ・進捗・明快な操作）は残す。将来は色トークンを差し替えて配色変更できる前提。
>
> 主役: ユーザーが目的ごとに作る「フォルダ」＋手動の単語登録（プリセットはサンプル1つだけ同梱）。
>
> 学習モデル: 受け身の「表示回数メーター」（信号風の3段バー）。見るほど 1本→2本→3本 と育ち、満タンで自動お休み（たまに復習で再登場）。タップ不要。正確な回数は単語詳細だけに出す。
>
> ウィジェット: 小/中/大＋ロック画面。各単語にメーター。音声（発音）ボタンあり。
>
> 画面: ホーム(Today) / フォルダ一覧 / フォルダ作成 / 単語登録 / 単語詳細 / オンボーディング(3ステップ) / 設定。ナビは英語ラベル(Today / Folders / Settings)。
>
> コンテンツは日本語、対象言語の例は韓国語＋カナ読み。
>
> お願い: 上のモノクロ方針を維持したまま、レイアウト・余白・文字サイズ・整列・トーンを整えてください。

## 守ってほしいこと（Do / Don't）
- Do: モノクロ、静か、余白広め、ヘアライン、角丸控えめ、操作部だけ濃いインク
- Don't: 差し色を足す、カラフルにする、バッジ/ランキング、テスト/クイズ感、学生っぽさ
