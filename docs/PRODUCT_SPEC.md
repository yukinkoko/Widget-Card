## 0. v1（6/19）→ v2（7/14）の主な変更
v2 デザインは v1 で「Phase 2 / 不採用」としていた要素を Phase 1 相当に引き上げている。以下を**確定として採用**する。

| 項目 | v1（旧） | v2（現・確定） |
|---|---|---|
| メーター | 信号風3段（Resting/Review） | **線形 n/10 → Learned**（10回で定着）。Word detail は10分割セグメント |
| 手動状態 | 「覚えた/まだ」不採用 | **「覚えた」あり**（詳細で即Learned）。用語は Learned / 覚え中 |
| AI生成 | Phase 2 | **Phase 1採用**（テーマ→候補生成→選択→一括追加） |
| 目標期限・推奨語数 | 不採用 | **採用**（1week/1month/3month/Date＋おすすめ語数） |
| 配色 | 完全モノクロ（accent=インク） | **アクセント緑 `#78FC90` ＋トーン3種**（color/dark/light） |
| 通知/CSV/iCloud | 最小 | **Settings に明記**（リマインダー通知・CSV書き出し・iCloud） |
| オンボーディング | 〜3ステップ | **5ステップ** |
| 対象 | iPhone優先 | **iOSのみ（当面）** |

## 1. コンセプト
- 一言: **覚えたい言葉を、毎日の画面に。**
- ホーム/ロック画面に単語・読み方・意味が常に表示され、**タップしなくても、ながら見で少しずつ覚えられる**。
- 「単語帳/テスト/クイズ」アプリではない。**毎日の画面に置いておく小さな言葉の学習アプリ**。

## 2. ポジショニング（主役）
- **主役 = ユーザーが目的ごとに作る「フォルダ」＋単語登録（AI生成 or 手動）**。
- プリセットは廃止し、**サンプルフォルダ**として同梱（空状態回避・使い方提示）。
- 目的が伝わる名前を推奨（例: 韓国旅行で使う単語 / 英語会議 / デザイナー英語）。

## 3. 対象ユーザー
- ライト学習層／**旅行前の実用層（P1の一番の狙い）**／ながら勉強層。
- 共通: 勉強したい気持ちはあるが毎日アプリを開く習慣はない。急かされるより自然に目に入る方が続く。

## 4. 差別化
- 常に表示・ながら見（クイズ型を主役にしない）／英語に限定せず**目的別フォルダを自分で作れる**／静か・ニュートラル。
- **AIで目的（テーマ）を伝えるとフォルダ＋単語ができる**（v2で P1 に前倒し）。

## 5. 技術構成・対象プラットフォーム
- **iOSのみ（当面）**。ロック画面ウィジェットは事実上 iOS。
- **Kotlin Multiplatform テンプレート活用**:
  - 本体アプリ = **Compose Multiplatform**
  - **ウィジェット = ネイティブ SwiftUI / WidgetKit**
  - 本体 ↔ ウィジェット = **App Group**
- ビルド/実行は Xcode。

## 6. デザイン言語
- 基調はモノクロ（インク `#111110` / 背景 `#EDEDEB` / カード `#FFFFFF`）だが、**アクセント緑 `#78FC90`** をメーター進捗・達成ドット・今日チップのドットに使用。
- **トーン3種**（color / dark / light）を全画面・ウィジェットに適用。Settings「外観 > アプリのカラー」で切替。
- 書体: 欧文 **Plus Jakarta Sans**、和文 **Noto Sans JP**、韓国語 **Noto Sans KR**。ウェイトで階層。
- **色はトークン化**: `accent` を1トークンに集約し差し替え可能に（将来のきせかえ拡張の余地）。
- 静か・大人っぽい・毎日見ても疲れない。ゲーム/カラフル/バッジ・ランキングは避けるが**「ちゃんとしたアプリ」の作り込み（カード・タブ・進捗・明快な操作）は残す**。
- 詳細トークン（色・タイポ・角丸・影の確定値）は `docs/IMPLEMENTATION_PLAN.md` §3 と HTML 一次情報。

## 7. 学習モデル（コア）= 受け身の表示回数メーター（n/10）
- **表示回数メーター**: 単語が画面に出る（出会う）たびに進む。**タップ不要**。バー＋数値「**n / 10**」。進捗色 `#78FC90`。
  - **10回で Learned**（定着）。Word detail のみ10分割セグメントで表示。
  - サーフェス（ウィジェット/リスト）はバー＋n/10。正確な回数の文言は Word detail に集約。
- **手動「覚えた」**: Word detail の白ボタンで**即 Learned 化**。詳細の操作語は「覚えた」「覚え中」。
- **Learned は表示ローテーションから自動除外**（Settings で「Learnedを表示から外す」オフ可）。
- しきい値（曜日チップの 10+ 判定なども含む）は調整可（暫定10）。

## 8. 用語・ラベル
- 学習コンテンツ（意味）は**日本語**。単語・読み方は対象言語（韓/英/中）。
- 状態語 = **Learned**（覚えた/覚え中）。
- UIナビ語は**英語**（Daily / Folders / Settings）。

## 9. ウィジェット（3サイズ＋ロック画面）
- **Small 170×170**: 単語 / 読み方 / 短い意味 / バー（＋音声）。2行省略。
- **Medium 360×170**: フォルダ名 / 単語 / 読み方 / 意味 / バー（＋音声）。長文は自動縮小。
- **Large 360×360**: 今日の複数単語（各: 単語 / 読み方 / 短い意味 / バー）。
- **ロック画面**: 単語 / 読み方 / 短い意味。
- **トーン3種**（color=白地×緑 / dark=`#1C1C1E` / light=白地×黒メーター）。
- **音声ボタン（P1）**: §10。
- **Widget設定**（長押し→編集シート）: フォルダ / カラー3セグメント / メーター・フォルダ名・読み方・意味トグル（ON=黒） / 再生ボタン。

## 10. 音声（P1）
- P1 からウィジェット上に音声（発音）ボタン。
- **ホーム画面**: ボタン → **App Intent** → `AVAudioSession(.playback)` → `AVAudioPlayer` でバンドル音声を再生（`.playback` はマナーモードでも鳴る）。
- **ロック画面**: 制約が強く「アプリを開いて再生」フォールバックの可能性 → 要検証。
- **進め方**: 実装初期に音声スパイクを1本作り検証（`IMPLEMENTATION_PLAN.md` M4a）。
- 音声アセットは事前生成してバンドル（P2で録音/AI生成に拡張）。

## 11. AI（テーマ→単語生成・自動補完）
「ずっと無料・上限なし・オンデバイス」を最優先。**Foundation Models は対応機種が狭い（iPhone 15 Pro+/iOS 26）ため不採用**。

- **意味の自動補完（翻訳）** → **Apple Translation framework**（`TranslationSession`, iOS 18+, 無料・無制限・オンデバイス・KR/JP/EN/ZH対応・iPhone XS以降）。実機必須・言語パックDL要。
- **テーマ→単語リスト生成／読み方（音写）** → **小型オンデバイスLLM**（MLX Swift or CoreML/ANE、Qwen2.5-1.5B / Gemma-2-2B級・量子化）。ランタイムは実装スパイクで確定。品質は翻訳程度で妥協可。
- どちらも Swift 専用API → **AIは iOS ネイティブ層に実装**、Compose からはインターフェイス越しに呼ぶ。
- **全機種で手動登録フォールバック必須**（AI非対応・言語パック未DL時）。
- AI生成フロー: テーマ＋言語＋語数 → 候補生成（約10秒・キャンセル可）→ チェックで取捨選択（n/28選択中）→ 一括追加。

## 12. データモデル（v2）
### フォルダ
| 項目 | 必須 | 用途 |
|---|---|---|
| id | 必須 | 識別子 |
| フォルダ名 | 必須 | 例: 韓国旅行で使う単語 |
| 説明 | 任意 | 何のためか（AIテーマの素にも） |
| アイコン | 任意 | 見分け（4タイルから選択） |
| 目標期限 | 任意 | 1week/1month/3month/Date。おすすめ語数の根拠 |
| 単語配列 | 必須 | 含まれる単語 |
| 表示中フラグ | 必須 | ウィジェット/Daily の対象 |

### 単語
| 項目 | 必須 | 用途 |
|---|---|---|
| 原語 | 必須 | メイン表示 |
| 読み方 | 必須 | 発音補助（カナ/ローマ字） |
| 意味 | 必須 | ウィジェット/リスト表示 |
| 出会った回数(0–10) | 必須 | メーターの素 |
| Learnedフラグ | 必須 | 10到達 or 手動で true |
| 表示順 | 必須 | ローテーション順 |
| 言語 | 必須 | KR/EN/ZH 等 |

### 集計・設定
- **日別ながら見回数**（曜日チップ状態: 0 / 1–9 / 10+）／Learned数・総数。
- **ウィジェット設定**: フォルダid / トーン(color・dark・light) / 表示項目トグル / 再生ボタン。
- **アプリ設定**: リマインダー(on/off・時刻) / アプリのカラー(color/dark/light) / iCloud / Learnedを表示から外す。

## 13. 画面構成
| 画面 | 役割 | タブ |
|---|---|---|
| Daily | 曜日チップ列 / 黒カード（フォルダ・n/28・TODAY+3・↗・バー） / TODAY WORD リスト | タブ1 |
| Daily・未設置 | ウィジェット未設置状態（手順3ステップ、CTAなし・自動切替） | タブ1 |
| Folders / 0件 | フォルダ管理・切替・新規作成 / 空状態＋作成入口3カード | タブ2 |
| Folder作成 | 名前 / 追加方法2カード / 目標期限チップ＋日付 / おすすめ語数 / アイコン4タイル | （遷移） |
| AI単語登録 / 生成中 | テーマ・言語・語数 → 候補生成 → チェック選択 → 一括追加 / 進行カード | （遷移） |
| Word entry | 単語・読み方・意味（✦自動入力）＋「＋もう1語追加｜追加」 | （遷移） |
| Word detail | Mediumヒーロー / トーン切替 / 出会った回数（10分割）/ 白ボタン「覚えた」 | （遷移） |
| Word list | タイトル＋「28 words · 12 Learned」/ All・Learned セグメント / 罫線表 / ＋単語を追加 | （遷移） |
| Settings | リマインダー / ウィジェット既定トーン・再生・Learned除外 / データ(iCloud・CSV) / その他 | タブ3 |

タブは **Daily / Folders / Settings** の3つ。アクティブ＝黒ピル（アイコン＋英語ラベル）、非アクティブ＝白丸46px。コンテンツ上にフェード帯で浮く。1階層下ではナビ非表示。

### オンボーディング（5ステップ・スキップは①②のみ）
①価値（3言語のMedium実UIを傾けて積む: 韓・英・中）→ ②仕組み（Medium実UI＋3ノードのタイムライン: 1回目→6回目 なじんできた→✓10回目 Learned）→ ③フォルダ作成（テーマ入力＋候補チップ）→ ④ウィジェット追加（スマホモック＋手順・「追加できた／あとでやる」）→ ⑤完了（サマリー2行）。

初回で聞かない: 学習目標 / 学習時間 / レベル診断 / 通知許可（先に体験）/ アカウント登録。

## 14. スコープ
**Phase 1（公開初期）= v2フル**
- フォルダ作成（AI生成 ＋ 手動登録）／サンプルフォルダ同梱
- Daily / ホーム＆ロック画面ウィジェット（3サイズ＋トーン3種）
- 表示回数メーター（n/10 → Learned・自動除外）／Word detail・Word list
- ウィジェット音声ボタン（要検証）
- AI: Apple Translation（意味）＋小型オンデバイスLLM（生成・読み方）
- 目標期限・推奨語数／リマインダー通知／CSV書き出し／iCloud
- オンボーディング5ステップ

**Phase 2 以降**
- 音声の拡張（録音/AI生成）／色テーマ拡張（きせかえ・フォルダ色）／AIランタイムの高度化

**対象外（初期）**
- ランキング / 友達共有 / 学習時間記録 / 大量テストモード / 複雑な復習間隔の手動設定 / PDF・画像・ノートからの生成 / アカウント必須化 / Android・macOS

## 15. エッジケース（実装時仕様メモ）
- **期限切れ**: 「あと0日」以降はバッジを「期限切れ」表示にし、期限再設定を促す。
- **フォルダ完走**（全語Learned）: お祝い表示＋次フォルダ作成への誘導（未デザイン・要追加）。
- **通知ベル**: Daily右上ベルの行き先は未定義（通知一覧 or リマインダー設定）。
- **編集・削除**: 単語行は左スワイプで削除、Word detail に編集導線を追加予定（未デザイン）。

## 16. App Store で伝えること
1. 覚えたい言葉を、毎日の画面に。
2. 単語・読み方・意味がいつも見える。
3. 旅行や仕事に合わせて**フォルダを作れる**（AIでも手動でも）。
4. ウィジェットで、アプリを開かずに復習できる。
5. 学習アプリっぽすぎない静かな見た目。
